Microsoft работает над «веб-ориентированным» вариантом Windows 12

Облачная версия Windows 12 не предназначена для замены традиционной версии Win32 на настольных компьютерах. Это будет новый вариант, адаптированный для конкретных рынков. Microsoft пытается создать более современную, модульную и лёгкую версию Windows для недорогих образовательных устройств. Компания уже подтвердила во внутренней записке, что формирует новую команду «Windows and Web Experience» для разработки следующих поколений ОС и веб-продуктов. Она планирует «определить и предоставить следующее поколение возможностей и приложений Windows, охватывающих клиентские и облачные среды».

Познакомьтесь со вторым пилотом в Windows

Найдите информацию и идеи, необходимые для развития вашей изобретательности. Второй пилот в Windows6 - это новая функция искусственного интеллекта в Windows 11. С ее помощью вы сможете быстро получать ответы и задавать дополнительные вопросы, создавать графику, созданную искусственным интеллектом на основе ваших идей, и развивать свой творческий потенциал во время работы. Познакомьтесь со вторым пилотом в Windows, вашим новым интеллектуальным помощником.

С Windows 11 мы наблюдаем рекордную вовлеченность, самую высокую удовлетворенность наших клиентов по сравнению с любой предыдущей версией Windows в США и ускоренный рост числа коммерческих внедрений.
